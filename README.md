# 🧠 Хекслет‑match — AI-платформа для подбора IT-команд

> **AI-рекрутинг нового поколения**  
> Автоматическая оценка hard skills + психологической совместимости (culture fit).  
> Сокращение времени закрытия вакансии с 30 до 7 дней, снижение затрат на найм до 40%.

---

## 📌 О проекте

**Хекслет‑match** — это минималистичный веб-сайт-презентация продукта, разработанный в рамках практической работы.  
Проект демонстрирует:

- актуальность AI-инструментов для IT-найма
- анализ конкурентов (GitWork, Хабр Карьера, Авито Работа)
- ёмкость рынка (TAM ≈ 3,24 млн руб./год по России)
- ценностное предложение для CTO, основателей стартапов и IT-рекрутеров
- MVP (прототип) с карточками кандидатов в стиле Tinder
- результаты PoC (50% готовности платить)
- North Star Metric (количество успешных наймов)
- A/B-тест гипотезы о влиянии culture fit на конверсию
- Customer Journey Map (CJM) для CTO
- вывод о целесообразности вывода на рынок
